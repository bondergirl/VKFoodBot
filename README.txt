Бот для ознакомления с ассортиментом еды по API ВКонтакте.

Установка:
- Токены бота и приложения ВК в файле окружения .env

Команды:
    /low - выведет заданное количество выбранной еды по возрастанию цены
    /high - выведет заданное количество выбранной еды по убыванию цены
    /custom - выведет заданное количество выбранной еды в заданном диапазоне цен
    /history - выведет историю запросов
